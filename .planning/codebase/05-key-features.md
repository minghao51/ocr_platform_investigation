# Key Features

## Multi-Provider VLM Support
- **Nebius**: Default provider
- **OpenRouter**: Multi-model support (OpenAI, Anthropic, etc.)
- **Google Gemini**: Gemini Pro/Flash models

## Auto-Routing
- Automatically selects optimal extraction method based on document type
- Digital PDFs → fast text extraction
- Scanned PDFs/images → vision models

## Schema-Based Extraction
- Built-in templates: Invoice, Receipt, ID, Generic
- Custom JSON schema builder
- Field types: Text, Number, Boolean, List of Text

## Processing Modes
- **Auto**: Automatic selection (recommended)
- **Text**: Fast PDF text extraction
- **Vision**: AI vision understanding
- **Hybrid**: Text + Vision validation

## Advanced Options
- Custom prompt instructions
- Temperature control (0-1)
- Max tokens (256-32768)
